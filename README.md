# Rajat Krishnan Portfolio

Static portfolio website for GitHub Pages at `https://rajat77a.github.io`.

## Rajat Intelligence

The portfolio includes a local, no-external-API assistant powered by
`knowledge.js`. It answers only from approved Rajat profile facts gathered from
the resume, GitHub profile, public repo metadata, and submitted links.

Normal answers are intentionally short. Unverified or unrelated questions are
refused instead of guessed.

## Deploy

Push the contents of this repository to the GitHub Pages repository or branch
that serves `rajat77a.github.io`.
