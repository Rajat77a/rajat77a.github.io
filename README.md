# Rajat Krishnan — Portfolio

Personal portfolio showcasing my work in AI, web development, and real products built on zero budget.

## Live Site

[rajat77a.github.io](https://rajat77a.github.io/)

## Pages

- **Home** — Hero intro, case study preview, contact links
- **PrepPeer** — AI mock interview platform case study with real product screenshots
- **About** — Background, skills, and contact

## Features

- **Mobile-first responsive design** — works on phones, tablets, and desktop
- **Accessible** — skip link, focus states, WCAG AA contrast, semantic HTML
- **Fast** — lazy loading images, optimized CSS, no heavy dependencies
- **Dark theme** — clean dark UI with teal accent color

## Built With

- HTML5, CSS3 (custom properties, clamp(), flexbox, grid)
- Google Fonts (Inter, Space Grotesk)
- GitHub Pages for hosting
- Deployed via GitHub Actions

## Accessibility

- Skip link for keyboard navigation
- `:focus-visible` outlines on all interactive elements
- WCAG AA color contrast (7.5:1 on body text)
- 44px minimum tap targets on all buttons/links
- Semantic HTML (`<main>`, `<nav>`, `<section>`)
- `aria-label` on interactive elements

## Performance

- Lazy loading on all images
- Font preconnect for faster font loading
- No JavaScript dependencies
- Minimal CSS (no frameworks)
